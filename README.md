# Tree Based Regression Models

This project implements and compares several tree based and ensemble machine learning algorithms using the Boston Housing dataset in Python. The objective is to evaluate the predictive performance of different models for estimating median home prices.

## Project Overview

The Boston Housing dataset was loaded and split into training and testing sets. Multiple tree based algorithms were trained and evaluated using common regression metrics.

The project compares individual decision trees with ensemble methods to determine which model provides the most accurate predictions.

## Models Implemented

* Decision Tree Regression
* Bagging Regression
* Random Forest Regression
* Extra Trees Regression
* Gradient Boosting Regression
* AdaBoost Regression

## Evaluation Metrics

* Root Mean Squared Error (RMSE)
* Mean Absolute Error (MAE)
* R Squared (R²)
